### **How to use**

- 1: Run<strong> yarn</strong> ( install all dependencies);
- 2: Run<strong> yarn react-native run-ios or yarn react-native run-android</strong> ( build app in emulator);
- 3: Run<strong> yarn start</strong> ( Run server );
